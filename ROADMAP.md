# Roadmap

## L0 (Launch)
- Public calendar list + search/filter
- Indicator detail pages with historical table
- Admin CSV upload (shared-secret)

## L1 (Next)
- User accounts + saved watchlist
- Email/webhook alerts for selected indicators
- Better categorization (regions, tags)
- Import pipeline from chosen sources (with licensing)

## L2 (Later)
- Instant release alerts + push/websocket
- Revision tracking and diff views
- Full API + rate limiting + billing
